# Let's SOLVE It Spring 2026 — Application Guide

## Important Warning

> **You will not be able to save the application once you start.** Prepare all materials before beginning the form.

## Application URL

https://rbcborealis.com/program-applications/lets-solve-it-spring-2026/

---

## Team-Level Submissions

### 1. Project Proposal (Required)

- **Length:** 500–1,000 words
- **Format:** PDF, DOC, DOCX, or RTF
- **Must address:**
  - The problem you want to solve
  - Why the problem matters to your team and/or community
  - Why machine learning could help solve it
  - What data you plan to use (2–3 potential datasets)
  - Whether the project relates to personal or extracurricular work
  - Whether the project fulfills coursework requirements (must disclose)
  - Willingness to pivot to an alternative community project if selected

### 2. Secondary Project Proposal (Optional)

- Same format as primary proposal
- Evaluated only if team is selected
- Mentor chooses between primary and secondary proposals

---

## Per-Member Fields (Collected for All 3–5 Members)

### Contact Information
- First name
- Last name
- Email address
- Phone number
- T-shirt/sweatshirt size (XS/Size 4 through XL/Size 14)

### Educational Details
- Canadian university/college name
- Department/program name
- Province/territory of university
- Year of study (Undergrad Year 1, 2, 3, 4, 5+)
- **Proof of enrollment** (transcripts or acceptance letter) — required

### Technical Skills (Open-Ended Text Field)
Document any of the following:
- Programming courses completed
- School projects involving programming
- Software development skills
- Online programming or AI courses (extracurricular)
- Hackathon or competition participation
- Related personal projects
- GitHub profile link

### Resume/CV
- **Team Captain:** Required
- **Other members:** Optional

### Diversity Self-Identification (Voluntary)
All optional:
- Aboriginal/Indigenous person status
- Visible minority status (categories: Arab, Black, Chinese, Filipino, Japanese, Korean, Latin-American, South Asian, Southeast Asian, West Asian, Mixed, Other, Unknown)
- Disability status
- Gender (Male, Female, Other, Non-binary/Gender nonconforming/Genderfluid/Intersex/Genderqueer/Agender)
- LGBTQ+ community membership

## Pre-Application Checklist

Before starting the form, have ready for **each team member**:

- [ ] Full name, email, phone
- [ ] University name, program, year of study, province
- [ ] Proof of enrollment document (transcript or acceptance letter)
- [ ] Technical skills summary written out
- [ ] Resume/CV (required for captain, optional for others)

And for the **team**:

- [ ] Project proposal document (500–1,000 words, PDF/DOC/DOCX/RTF)
- [ ] Optional: secondary project proposal
- [ ] Confirm all members can attend Welcome Day and Presentation Day
- [ ] Confirm weekly availability (2+ consecutive hours Mon–Fri 9am–5pm EST)
